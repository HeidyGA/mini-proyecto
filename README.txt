# Mini Proyecto - Análisis de Encuesta

Este proyecto procesa los datos de una encuesta estructurada por temas y preguntas. Calcula promedios, medianas, modas, extremismo y consenso, y muestra la información ordenada como lo exige un cliente simulado.

## Estructura del Proyecto

- `main.py`: archivo principal de ejecución
- `modelos.py`: clases `Encuestado`, `Pregunta` y `Tema`
- `cargar_datos.py`: función para leer archivos de entrada
- `utilidades.py`: funciones auxiliares para métricas y ordenamientos
- `datos_ejemplo.txt`: archivo de entrada de prueba
- `README.txt`: documentación del proyecto
- `segunda_solucion.py`: la segunda solución alternativa

## Ejecución

```bash
python main.py
python segunda_solucion.py
```
